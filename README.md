# ⛅Weather-App
Простая консольная утилита для получения текущей погоды в указанном городе. Скрипт использует API `OpenWeatherMap` и автоматически переводит описание погоды на русский язык.
________

## 🚀 Инструкция по установке
1. **Клонируйте репозиторий** или скачайте файл со скриптом.
2. **Установите необходимые зависимости**:
   - Скрипт использует библиотеки `pyowm`, `python-dotenv`, `asyncio` и `googletrans`. Установите их командой:
```
  pip install pyowm python-dotenv googletrans asyncio
```

## ⚙️ Настройка (API Ключ)
Программа считывает API ключ из файла окружения.
1. Создайте в корневой папке проекта файл с названием `.env`.
2. Добавьте в него следующую строку, подставив свой ключ:
```
  OpenWeatherMapApi = ВАШ_API_КЛЮЧ
```

## 💻 Использование
Для запуска программы передайте название города в качестве аргумента командной строки:
```
  python weather.py "Moscow"
```
**Пример вывода:**
```
  Город: Moscow
  Температура(°C): -5
  Описание погоды: небольшая облачность
```

## 📂 Структура проекта
+ `weather.py` — основной код программы.
+ `.env` — файл с вашими секретными ключами.
+ `README.md` — инструкция, которую вы сейчас читаете.

## ⚠️ Возможные ошибки
+ `NotFoundError`: Проверьте, правильно ли написано название города.
+ `UnauthorizedError`: Убедитесь, что ваш API ключ в файле `.env` активен и скопирован без лишних пробелов.
+ `Ошибка перевода`: Возникает при проблемах с доступом к серверам Google. В этом случае скрипт выведет описание на английском.
